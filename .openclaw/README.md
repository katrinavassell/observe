# OpenClaw Config — Observe

Slack channel `agent-monetization` (C0B4DV1D1V2) is configured here.

## Setup

1. Copy `openclaw.json` to your OpenClaw instance's config
2. Set `botToken` and `appToken` from your Slack app (not committed for security)
3. Restart the gateway

Tokens are intentionally excluded. Set them via environment variables or add them to the config on the target machine.
